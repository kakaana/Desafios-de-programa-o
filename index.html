Desafio do Dia: Contador de Vogais

Nível: Iniciante | Tópico: Strings

Enunciado do Problema

Crie uma função que receba uma string (palavra ou frase) e retorne o número total de vogais (a, e, i, o, u) que ela contém, desconsiderando a capitalização.

Exemplo:

Entrada: "Programacao"

Saída Esperada: 5

Sua Solução (Pseudocódigo)

function contarVogais(str) {
    // Converte para minúsculas para simplificar a contagem
    str = str.toLowerCase();
    let contador = 0;
    const vogais = 'aeiou';

    for (let char of str) {
        if (vogais.includes(char)) {
            contador++;
        }
    }
    return contador;
}

// Teste:
console.log(contarVogais("Programacao")); // Saída: 5
